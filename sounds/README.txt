  Happy Paws — HOW TO ADD YOUR OWN ANIMAL SOUNDS
════════════════════════════════════════════════════════

This folder is where you place your custom sound files.

SUPPORTED FORMATS:  .mp3  .ogg  (MP3 recommended)

STEPS:
  1. Find or record your animal sound
     (e.g. a real dog bark, cat meow, parrot squawk…)

  2. Copy the audio file into THIS folder (sounds/)
     Example: sounds/luna_bark.mp3

  3. Go to the Admin Dashboard → Edit the animal
     In the "Custom Sound File" field, type ONLY the filename:
         luna_bark.mp3
     (Do NOT include the  sounds/  path — just the filename)

  4. Save. Done! The site will now play your file on hover.

TIPS:
  - Short clips work best (0.5 – 2 seconds)
  - Lower file size = faster load (aim for < 100 KB)
  - Free animal sounds: freesound.org, zapsplat.com
  - If the file is missing, it silently falls back
    to the built-in synthesized sound.

BUILT-IN FALLBACK SOUNDS (no file needed):
  🐕 Dog     → Bark
  🐱 Cat     → Meow
  🐇 Rabbit  → Squeak
  🐹 Hamster → Squeak
  🐦 Bird    → Tweet
  🦜 Parrot  → Squawk
  🦅 Eagle   → Screech
